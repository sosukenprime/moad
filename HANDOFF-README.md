# MOAD Handoff Package

Welcome. You're here because you're porting MOAD from a Claude artifact to a real app.

## Read in this order

1. **`CHECKLIST.md`** — step-by-step setup instructions. Start here. ~30 min setup.
2. **`MOAD-HANDOFF.md`** — the full project spec. Read before writing any code.
3. **`CLAUDE-CODE-PROMPT.md`** — the prompt to paste into Claude Code when you start.

## Reference files (Claude Code will read these)

4. **`moad-v2-reference.html`** — the working prototype. Don't port line-by-line — use as reference for behavior and styling.
5. **`MOAD-brief.md`** — original project brief with research findings and rationale.
6. **`design-config.json`** — locked design choices in machine-readable form.

## Quick tip

Keep all 6 files in the SAME folder when you start Claude Code. It reads from the folder you launch it in.
